# 3. Hafta: Uygulama Geliştirme, Git Kullanımı ve Docker Entegrasyonu

Bu hafta, teorik bilgilerin pratiğe dökülmesi ve versiyon kontrol sistemlerinin (Git) iş akışına dahil edilmesi üzerine çalışılmıştır.

### Yapılan Çalışmalar:
- **Kod Geliştirme:** Docker üzerinde çalışan PHP tabanlı başlangıç projesi oluşturuldu (`index.php`).
- **Git & GitHub:** Proje dosyaları VS Code terminali üzerinden Git komutları (`init`, `add`, `commit`, `push`) kullanılarak GitHub reposuna aktarıldı.
- **Discord Otomasyonu:** Uygulama içi hata ve sorunlar konuşuldu.
- **Versiyon Kontrolü:** "Main branch" yapısı ve commit mesajı standartları uygulanarak profesyonel bir repo hiyerarşisi oluşturuldu.

### Kazanılan Teknik Yetkinlikler:
- Terminal üzerinden Git yönetimi.
- PHP servislerinin Docker konteynerleri ile uyumlu çalıştırılması.
### Kullanılan Uygulamalar
-Docker
-VS Code
-Git-Github
